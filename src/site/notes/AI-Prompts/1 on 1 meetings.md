```
This is a one-on-one meeting [with my manager] / [with a person I manage], please create meeting notes with the following structure:

**Meeting Details:**
- Date and participants

**Performance and Progress:**
- Achievements that were specifically mentioned
- Challenges that were discussed
- Goal progress updates that were shared

**Development Discussion:**
- Career goals and aspirations that were brought up
- Skill development needs that were identified
- Training or learning opportunities that were discussed
- If development wasn't discussed, state "Development topics not covered"

**Feedback Exchange:**
- Specific feedback that was given to employee
- Feedback that was received from employee
- Areas for improvement that were mentioned
- If no formal feedback was exchanged, state "No formal feedback exchanged"

**Support Needed:**
- Support that was specifically requested
- Obstacles that were identified
- Manager commitments that were made

**Action Items:**
- Development activities that were agreed upon. Include timelines if mentioned
- Follow-up items for both parties that were established
- Only include actions that were explicitly discussed and assigned
```
